How to use Commands?

Find out how to use B-MoBot!

Our Prefix is: " b! "

Public Commands
  • b!server. (To display main server info, like Members, Roles, Channels, server ID & others)
  • b!info. (To get the general info's about B-MoBot, like Bot status, Version & more!)
  • b!user. (Request your account info, or others accounts info's)
  • b!avatar. (Display your account avatar, or other's avatar)
  • b!savatar. (To show the avatar of the current server)
  • b!rules. (Submit the B-MoBot's rules)
  • b!ping. (Bot's Ping)
  • b!inv. (Invite the Bot to your server!)
  • b!afk. (To set you AFK)
Games Commands
  • b!speed. (The fastest person to type random words)
  • b!puz. (Intelligence game for the fastest person to solve the puzzle for 15 seconds)
  • b!fkk. (The fastest person to decipher random words)
  • b!fkl. (Frankly speaking, there is no timing)
  • b!cut. (Cut tweets game, without time)
  • b!slo. (A game you and your luck and hard to win)
  • b!flags. (Flags Game You have 15 seconds for the fastest person to write the name of the country)
  • b!math. (Math game you have to solve for the correct result)
  • b!xo. (The XO game can be played with only two people)
  • b!aki. (blue almard game)
